# Manual del repartidor

App: **driver-pwa** (móvil / PWA)

## Onboarding

1. Registro
2. Datos personales (`/onboarding`)
3. Documentos (`/documents`)
4. Vehículo (`/vehicle`)
5. Enviar solicitud (`/application`)
6. Esperar aprobación del admin

## Trabajo diario

1. Instalar PWA (si el navegador lo permite)
2. Activar disponibilidad (GPS)
3. Ver pedidos en `/jobs` y ofertar
4. Gestionar entrega en `/delivery` (estados + evidencia foto)
5. Revisar billetera `/wallet`
6. Preferencias de notificación en `/profile`
7. Soporte en `/support` si hay incidencias

## Evidencia fotográfica

Antes de marcar **Entregado**, subir foto (Fase 17). Sin foto el avance puede rechazarse.
