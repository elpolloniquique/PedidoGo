# Manual de instalación PWA (repartidor)

## Android (Chrome)

1. Abrir la app del repartidor en Chrome
2. Menú → **Instalar aplicación** / **Agregar a pantalla de inicio**
3. Confirmar

## iOS (Safari)

1. Abrir en Safari
2. Compartir → **Agregar a pantalla de inicio**
3. Confirmar

## Comportamiento

- Si ya está instalada, el aviso de instalación no se repite
- Offline: banner de estado; el perfil puede cachearse
- GPS solo con la app en uso y disponibilidad activa

## Limitaciones

- iOS no soporta el mismo prompt de instalación que Android
- Background GPS continuo depende del SO y del navegador
